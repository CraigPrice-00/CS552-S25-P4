# CS552 P4 - Bounded Buffer

Steps to configure, build, run, and test the bounded buffer.

## Building

```bash
make
```
## Build Debug version

```bash
make debug
```
## Testing

```bash
make check
```
```bash
./myprogram
```
## Clean

```bash
make clean
```

## Install Dependencies

In order to use git send-mail you need to run the following command:

```bash
make install-deps
```
